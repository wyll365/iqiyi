# 爱奇艺登陆接口
爱奇艺验证码破解,爱奇艺登陆,爱奇艺验证码

提供爱奇艺登陆接口,过滑块验证码

使用方法简单,一行代码即可返回爱奇艺cookie

详细见demo 例子
<pre>
    Api api=new Api();
   
    api.getDfp();// 爱奇艺dfp指纹
    
    api.decodeCaptcha();//滑块验证码 
    
    api.sendSms("13800138000");//发送短信验证码
    
    api.smsLogin("13800138000","111111");//短信登陆
    
    api.login("13800138000","123456");//账号密码登陆
    
</pre>
